Purpose: Take in binary contents from camera.o, then print out the contents with "." and "*", which represents 0 and 1 by the row and col given. Then pack and encode the data (separately) and print out the representation with "." and "*". Do again for another set of data.

Files:
-README.txt: Contains description
-a2-posted.c: Main program file
-camera.o: Holds data for a2.posted.c

Compiling and Running:
- Go to the directory that contains the program
- Type "gcc -o a2 a2-posted.c camera.o" to create a executable file
- Type "./a2" to run the executable file

Usage:
- Will print out contents in terminal
